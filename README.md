# 求职行动清单 · 产品作品集

独立于微信小程序工程的响应式产品 Case Study 网站。界面图来自项目 Figma 设计稿，网页结构、设备框、内容、布局与动效均由 React/CSS 实现。

## 启动

```bash
cd portfolio-site
npm install
npm run dev
```

## 构建与预览

```bash
npm run build
npm run preview
```

构建产物位于 `dist/`。页面内容集中在 `src/main.tsx`，视觉样式位于 `src/styles.css`，Figma 界面素材位于 `public/screens/`，便于后续替换。
